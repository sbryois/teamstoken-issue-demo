# teams-auth-issue

## Summary

Quick SPFx web part scaffolded with Yeoman connecting to demonstrate an issue with Token Acquisition when embedding a SharePoint site using this webpart within Teams.

Reproduction steps can be seen in the video under `demo_video`
[Video here](https://github.com/sbryois/teamstoken-issue-demo/tree/main/demo_video/demo.mp4)
